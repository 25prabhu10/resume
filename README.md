# Resume

Build using the [data-science-tech-resume-template](https://www.overleaf.com/latex/templates/data-science-tech-resume-template/zcdmpfxrzjhv)

## Files

- `resume.tex`: Main file
- `_header.tex`: Header code
- `TLCresume.sty`: Style file containing formatting details
- `section/objective.tex`: Objectives ([Examples](https://www.indeed.com/career-advice/resumes-cover-letters/resume-objective-examples))
- `section/skills.tex`: Table of skills
- `section/experience.tex`: Projects or roles
- `section/education.tex`: Schools and degrees
- `section/certifications.tex`: Certifications
